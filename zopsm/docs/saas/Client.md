#### Client

Client is the application device pair of a `consumer`.

__Attributes__:

| attr        | description                                      |
| ----------- | ------------------------------------------------ |
| consumer_id | unique identifier of `tenant`'s user.            |
| device_info | key value device information                     |
| push_tags   | tags for push service                            |
| sms_tags    | tags for sms service                             |
